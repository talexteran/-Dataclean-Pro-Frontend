# Dataclean Pro - SaaS de Limpieza de Datos

## Project Overview

- **Project Name**: Dataclean Pro
- **Type**: React + TypeScript SaaS Landing Page
- **Purpose**: Landing page para un SaaS de limpieza de datos
- **Target Users**: Empresas y profesionales que necesitan limpiar y normalizar datasets

## Features Implemented

- **Hero Section**: Mensaje principal con estadísticas de usuarios y registros procesados
- **Características**: 6 funciones principales del producto
- **Cómo funciona**: Proceso en 3 pasos
- **Planes de precios**: 3 niveles de suscripción
- **Call to Action**: Sección final de conversión
- **Footer**: Enlaces y copyright

## Design

- **Theme**: Dark mode con acentos cyan/blue
- **Typography**: System fonts con Tailwind
- **Color Palette**: Slate 900 (background), Cyan 400 (accent), Blue 600 (primary)
- **Effects**: Gradientes, backdrop blur, animaciones hover

## Tech Stack

- React 18 + TypeScript
- Vite 7
- Tailwind CSS 3
- No backend requerido (landing page estática)

## Build Commands

```bash
pnpm install
pnpm run build
```
